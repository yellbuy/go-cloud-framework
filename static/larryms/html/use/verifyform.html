forms组件验证相关简述:

larryms所封装的forms组件默认包含验证规则及方法描述：
1、自定义独立验证器：使用lay-vaild="blur|focus" 支持自定义回调 ,失去焦点验证支持自定义验证次数控制
2、提示方式： lay-verType="tips|alert|anim|below|other" 任选其一
3、内置规则：phone tel email url date uname int qq min max acepted between number identity aplpha alphaNum chs chsAlpha ip pwd pwdlevel isRepeat:idName isCreditCode isBankCard等等50多个（会根据需求陆续增加以满足开发需要）
4、自定义错误文本：lay-error="自定义内容1|自定义内容2" 按照验证规则顺序一一对应，其中使用默认提示的可以留空 。具体详细使用参考vip文档
