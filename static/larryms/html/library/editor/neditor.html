LarryMS框架之neditor编辑器组件封装

neditor编辑器简介：neditor编辑器是一款基于ueditor二次开发的现代化的富文本编辑器，它来自于github（https://github.com/notadd/neditor，若不需要在LarryMS框架中运行，用户可自行下载其开源版本，larryMS框架的收费服务与开源库无关），因其对ueditor进行了外观和功能的改进，LarryMS框架中将其进行符合layui规范的封装，可于框架内简单使用，为用户多一种选择方案，下面是在LarryMS框架中的使用示例：

neditor在ueditor基础上支持各种后端或者云存储、支持各种后端或者云存储、纯 ajax方式提交等新特性
larryms官方社区提供所有所提供扩展组件的维护和更新。当前版本：version:v2.1

ueditor编辑器【Code加载示例】 
layui.use('neditor', function() {
    var neditor = layui.neditor;

    var ue = UE.getEditor('editor');
    ue.ready(function() {
        ue.setHeight(200);
    });
});
//neditor是基于ueditor编辑器二开，这两款编辑器功能都非常丰富，如果只是简单应用也是一种不错的选择
//但其文档的繁杂会有不少坑需要去踩，ueditor和neditor的文档地址：http://fex.baidu.com/ueditor/#start-start
//larryMS框架中针对ue和ne这两款编辑器的封装只是让其加载使用更简便
