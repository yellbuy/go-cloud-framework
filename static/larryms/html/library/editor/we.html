LarryMS框架之wangEditor编辑器封装

wangEditor编辑器简介：wangEditor来源于github开源编辑器，一款基于javascript和css开发的Web富文本编辑器，轻量、简洁、易用、开源免费，LarryMS框架中对其进行了二次开发封装使其符合layui模块化加载规范成为一个模块化组件惰性加载，并进行了一些扩充【如内置功能新增防XSS过滤功能、代码高亮、全屏、响应式自适应、字体颜色、背景、code等等】，项目中应用更加方便高效，下面是wangEditor在LarryMS框架中的使用示例：

LarryMS框架之wangEditor - 轻量级web富文本编辑器，配置方便，使用简单。支持IE10 +和现代浏览器。
larryms官方社区提供所有所提供扩展组件的维护和更新。当前版本：version:v3.1.1 【推荐使用】

wangEditor编辑器【Code加载示例】 
layui.use('wangEditor', function() {
   var wangEditor = layui.wangEditor,
       editor = new wangEditor('#editor');
       editor.customConfig.uploadImgShowBase64 = true   // 使用 base64 保存图片
       editor.customConfig.uploadImgServer = '/upload'  // 上传图片到服务器
       editor.create();
});
