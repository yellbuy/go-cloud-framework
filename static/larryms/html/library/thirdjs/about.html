关于LarryMS框架所封装的第三方JS库重要说明

为了快速提高开发效率，LarryMS框架扩展封装了大量优秀的Javascript组件库、动画库，以实用为初衷降低学习成本提供开发效率为目的，紧贴业务需求，从底层进行编写实现与layui自带组件贯通，让您和使用layui官方组件一样使用，无任何延迟，如同layui自带组件一样使用且无需担心因layui版本更新而存在覆盖或更新等问题，扩展组件库与Layui目录并列存在于LarryMS框架系统中，LarryMS框架对所封装JS库提供更新和必要的技术支持，并且提供专属的使用文档示例教程

补充说明：关于封装的第三方收费JS库说明：未存在盗版或侵权，举例：比如极验官方提供的极验客户端js，LarryMS是对其进行了封装，让其更易于开发环境使用，简化操作省去页面引用与layui结合的更彻底，原接口还是需获得极验官方提供的接口（LarryMS只是作为一种提高开发效率的解决方案）。

关于第三方js库的封装，在LarryMS社区会给到大家封装教程，LarryMS框架中扩展了如：swiper、音频、视频、地图、百度统计、极验、区域打印、二维码生成、加解密、编码、excel表格、表格导入导出、水印、ztree、裁剪、剪贴板、多选select2、穿梭框、图片加载、echarts、css3动画库等等 多达60多个非常实用的第三方JS库,对这些优秀的Javascript库的使用LarryMS是作为服务提供,进行结合layui的封装扩展，对于来自github的开源库，larryMS框架中只是对其二次开发封装不涉及侵权，也不会将这些开源库用作商业收费，只是提供框架内对这些开源库进行二次开发使其符合LarryMS框架规范，模块化使用并提供服务，因此大家可放心使用，并提供持续更新与优化（可在larryMS社区反馈建议）

所有组件依赖已在底层解决和优化。使用举例：(如下所示无需extend别名，无需担心Jquery依赖等问题，就如同使用layer,form组件一样，撸起袖子就是干，方便直接，简单更高效) 

layui.use(['jquery','qrcode','ckplyer','geetest'],function(){
    var $ = layui.$,
        qrcode = layui.qrcode,
        ckplayer = layui.ckpalyer,
        geetest = layui.geetest; //无需在html页面中再去手动引入geetest.js文件
    //如ckplayer视频播放
    var vUrl = 'http://www.larryms.com/static/larrycms/video/larry.mp4',
        videoObject ={
                container: '#video',
                loop: true,
                autoplay: true,
                video:[
                    [vUrl, 'video/mp4']
                ]
    };
    var player = new ckplayer(videoObject);
    //qrcode使用
    var qrcodes = new qrcode(document.getElementById("qrcode"), {
        width : 300,
        height : 300
    });
    $('#product').on('click',function(){
        var url = $('#videoUrl').val().trim();
        qrcodes.makeCode(url);
        return false;
    });
});
