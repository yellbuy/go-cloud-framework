LarryMS框架扩展之pdf组件【提供于标准版及以上】

pdf组件：pdf阅读组件是LarryMS框架封装来自于GitHub的PDFObject轻量级JS库，用于在HTML文档中动态嵌入PDF，支持IE9-11，MS Edge，Safari，Firefox，Chrome。备注：PDFObject不是渲染引擎。PDFObject只是将一个 embed元素写入页面，并依赖于浏览器或浏览器插件来呈现PDF。如果浏览器不支持嵌入的PDF，则PDFObject无法强制浏览器呈现PDF【移动端部分浏览器不支持pdf嵌套，可以等larryms框架后续更新pdf组件】。LarryMS框架后续更新中会提供Mozilla的PDF.js功能更全面支持自定义工具栏；下面是PDFObject库封装后使用示例：

larryms官方社区提供所有所提供扩展组件的维护和更新。当前版本：PDFObject v2.0

#pdf阅读组件相对使用很简单，正如左侧预览效果，让浏览器支持PDF文件展示
layui.use('pdfobject',function(){
    var pdfobject = layui.pdfobject;
    //#没错就是这么简洁，一行代码实现pdf文件阅读，打印，常规操作如：全屏预览、下载、打印、缩放，该组件可用于一般浏览器pdf嵌套显示，更强劲丰富的功能可等待LarryMS框架后续提供的PDF组件。
    pdfobject.embed('./resource/sample-3pp.pdf','#pdfBox');
});
//CSS 支持
