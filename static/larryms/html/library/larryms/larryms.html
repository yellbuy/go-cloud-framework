框架核心组件之 larryms.js

larryms组件：作为LarryMS框架最核心的基础组件，用于支撑整个框架的运行，封装了非常多的常用基础功能，如：新老Jquery三方插件智能加载、多终端智能设备检测判断、表单独立验证扩展，密码复杂度检查中文验证、身份证验证、消息通知、十六进制颜色值、对象/数组类型检查与验证，字符串长度计算、HTML过滤、大小写转换、数组常用功能封装（排序/去重/遍历/包含检测/求和/最大最小值等等）、日期、数字金额格式转换、URL处理等等，以达到简化日常开发基础功能的复用

larryms组件方法介绍

名称 参数 用法 说明
plugin jsName,[callback],[jsConf]
描述：插件名称，回调函数，自定义配置项 		larryms.plugin('jsupersized.min',supersized);//插件名称可省略.js后缀也可不省略 如加载supersized插件，即登录页所看到的多背景图切换效果插件，智能灵活的方式更方便
set options
描述： options 参考开发者文档介绍 		larryms.set(options) 可以对框架相关核心参数做自定义，如弹窗提示标题自定义，框架名称修改、第三方Jq插件默认目录位置修改、动画库修改、字体默认设置等等
fontset configs
描述： configs 参考开发者文档介绍		 larryms.fontset({...}) 譬如：设置自定义扩展字体（教程可larryms官网社区中找到），默认larry-icon字体修改
notice options
描述： options 参考开发者文档介绍		 larryms.notice(options) 用于桌面堆叠通知，消息自定义位置主题等通知，支持语音提示
message msg，[mark],[icon],[time] larryms.message(msg) 这是一个较为简单的消息弹窗提醒，一般用户ajax前后台交互消息提示
typeFn 如isString isNumber isFunction isNull isUndefined isObj isArray isRegExp isSymbol isIOS isPC browserType等等具体参看开发者文档 larryms.typeFn.isArray(); larryms.typeFn.browserType(); larryms.typeFn.isObj()等等 typeFn为类型检测常用小功能类封装，该方法涉及功能就较为多了，可以简化一些基础工作，开发更高效
stringFn trim、strLenCount、changeCase、checkPwd、filterHtml等等，具体参看开发者文档 larryms.typeFn.trim(str, type); larryms.typeFn.checkPwd(str); stringFn 该方法的封装对于日常开发是很有用的，如trim() 虽然js有该方法，但对于低版本浏览器如IE 则不支持，因此larryms框架弥补了这类问题
ArrayFn contains、each、formArray、max、min、sum、average、remove、sort、map、unique等等，具体参考开发者文档 larryms.ArrayFn.contains(arr, val);//判断一个元素是否在数组中
larryms.ArrayFn.formArray(arr);//将类数组转换为数组的方法		 关于数组常用功能封装，也是为了简化基础繁琐的功能书写，让开发更高效
utils fixbar、getUrlParams、getUrlAllParams、delParamsUrl、random、numberToChinese、changeToChinese等等 larryms.utils.random(min, max)；//获得随机数，
larryms.utils.changeToChinese(Num);//将数字转换为大写金额		 larryms组件之常用工具类封装，可以如财务数字金额互相转换，一句话即可搞定
fullSize 无 larryms.fullSize() 该方法应用场景较为单一，主要是针对iframe页面在特殊情况下的铺满窗口高度设置
isCardID cardID larryms.isCardID('xxxx');//身份证号码 用于身份证号码格式检查以及身份证号码中相关信息智能识别返回，如地区、生日、地区、性别等等 功能也较为实用
verify 无 该方法无需手动执行框架会自动加载，用于对layui表单的验证扩充，当然也可以给其自定义加入新的验证规则，默认已加入密码强度和常规格式校验、邮政编码、qq号码、中英文、财务金额、银行卡号、整数、电话座机等验证 larryms.verify是验证规则的扩展，同类型的还有独立验证，表单中使用larryms-verify 用于单个表单中可以参看文档更新内容
更多功能,可在开发者文档中查看,同时也会根据VIP用户需求不断增加

JS代码【示例】

1、使用plugin方法加载第三方jq插件，代码示例

larryms.plugin('supersized.min',supersized);//加载supersized插件
function supersized(){
    $.supersized({
        slide_interval: 3000,
        transition: 1,
        transition_speed: 1000,
        performance: 1,
        slide_links: 'blank',
        slides: [{
                image: layui.cache.imgPath+'1.jpg'
            }, {
                image: layui.cache.imgPath+'2.jpg'
            }, {
                image: layui.cache.imgPath+'3.jpg'
            }]
        });
}
